## Công Nghệ Cần Đặc Biệt Chú Ý

### Frontend

- **React**: Kiểm tra phiên bản mới nhất, React Server Components, Concurrent Features
- **TypeScript**: Phiên bản mới nhất với type system improvements
- **Build Tools**: Vite, Webpack, hoặc alternatives mới hơn
- **State Management**: Zustand, Jotai, Redux Toolkit, hoặc built-in React features
- **UI Libraries**: shadcn/ui, Radix UI, Material UI - kiểm tra latest versions

### Backend & APIs

- **Node.js**: LTS version mới nhất trong 2026
- **API Standards**: REST, GraphQL, tRPC - best practices hiện tại
- **Authentication**: OAuth 2.1, OIDC, WebAuthn standards

### DevOps & Tools

- **Package Managers**: npm, pnpm, yarn - phiên bản và features mới nhất
- **Testing**: Vitest, Jest, Playwright - modern testing approaches
- **CI/CD**: GitHub Actions, GitLab CI - current best practices

## Lưu Ý Đặc Biệt

### Deprecations

- Luôn kiểm tra xem methods/APIs có bị deprecated không
- Tìm alternatives được khuyến nghị
- Plan migration path nếu cần

### Security

- Áp dụng security standards mới nhất (OWASP Top 10 2026)
- Sử dụng secure dependencies (kiểm tra vulnerabilities)
- Follow security best practices cho năm 2026

### Performance

- Áp dụng performance optimization techniques hiện đại
- Sử dụng modern browser APIs và features
- Consider Web Vitals và Core Web Vitals metrics
