# @shared/types

Shared TypeScript type definitions for the TPB CRM Platform.

## Usage

```typescript
import { 
  AgentStatus, 
  ChannelType, 
  Interaction,
  Customer,
  Ticket 
} from '@shared/types';
```

## Types Included

- **Agent Status Types**: AgentStatus, ChannelType, ChannelStatus
- **Interaction Types**: Interaction, InteractionStatus, InteractionPriority
- **Customer Types**: Customer
- **Notification Types**: Notification, NotificationType
- **Ticket Types**: Ticket, TicketStatus, TicketPriority
- **Common Types**: ApiResponse, PaginatedResponse, PaginationParams

## Development

This package contains only TypeScript type definitions and interfaces.
No runtime code is included.
